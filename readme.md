### 1.
    install nodejs
### 2. 
    trong thư mục chứa code chạy đoạn lệnh sau trên terminal
```bash
    npm install    
```
### 3.
    đợi đến khi chạy xong. vào file index.js chỉ lại đường dẫn  và số page 
### 4.
    chạy
```bash
    node index
```
    đợi đến khi chạy hoàn tất
### 5.
    chạy
```bash
    node download
```
### 6.
    sau khi hoàn tất . tất cả ảnh nằm trong mục download
